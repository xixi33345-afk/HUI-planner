# 上传 GitHub + 开通 Pages 傻瓜教程

全程在浏览器里点，不用装任何软件，约 10 分钟。照着做，每一步都告诉你点哪里。

---

## 准备：注册 GitHub 账号（有账号可跳过）

1. 打开 https://github.com
2. 右上角点 **Sign up**（注册），按提示填邮箱、设密码、起用户名
3. 邮箱里会收到验证邮件，点验证链接完成注册

> 记住你的**用户名**，后面网址会用到。

---

## 第一步：新建一个仓库（放代码的地方）

1. 登录后，点页面**右上角的 ➕** → 选 **New repository**（新建仓库）
2. 在 **Repository name** 框里输入仓库名，例如 `health`（只能用英文字母、数字、横杠）
3. 下面选 **Public（公开）** —— ⚠️ 必须是 Public，否则免费版不能开 Pages
4. 其它都不用动，拉到最下面点绿色按钮 **Create repository**（创建仓库）

---

## 第二步：上传你的文件

创建完会跳到一个空仓库页面，找到中间那行小字：
**"uploading an existing file"**（上传已有文件），点它。
（如果没看到，就把网址改成 `https://github.com/你的用户名/仓库名/upload/main` 直接回车）

### 关键：要上传"文件夹里的内容"，不是文件夹本身

1. 打开你电脑上的 `combined-app` 文件夹
2. **全选里面所有东西**（Ctrl+A）：包括 `index.html`、`manifest.json`、`sw.js`、`icon*.png`、`css` 文件夹、`js` 文件夹、`docs` 文件夹等
3. 把它们**整个拖进**网页中间那个大方框（写着 "Drag files here" 的区域）
4. 等下方文件列表加载完（有 `css/`、`js/` 子文件夹说明结构对了）
5. 拉到页面最下面，点绿色按钮 **Commit changes**（提交更改）

> 💡 不用上传 `启动网站.bat`（那是本地用的，上传了也无害，可留可删）。

---

## 第三步：开通 GitHub Pages

1. 进入你的仓库主页，点上方菜单栏最右边的 **⚙ Settings**（设置）
2. 在左侧菜单往下找，点 **Pages**
3. 看 **Build and deployment** 区域：
   - **Source** 选 **Deploy from a branch**（从分支部署）
   - 下面 **Branch** 左边的下拉框选 **main**，右边的下拉框保持 **/ (root)**
   - 点 **Save**（保存）
4. 保存后页面顶部会出现一句：
   **"Your site is live at https://你的用户名.github.io/仓库名/"**
   （如果暂时是 "Your site is being built"，等 1–2 分钟刷新一下就有了）

---

## 第四步：手机访问 + 加到主屏

1. 把上面那个 `https://你的用户名.github.io/仓库名/` 链接**发到微信文件传输助手**或自己的手机
2. 手机浏览器打开链接 → 网站就在公网上了，**任何网络都能访问**
3. 想当 App 用：
   - **iPhone（Safari）**：底部分享按钮 → 添加到主屏幕
   - **安卓（Chrome）**：右上角 ⋮ → 添加到主屏幕 / 安装应用
   - 主屏会出现紫色心形图标，点开像独立 App

---

## 以后怎么更新网站？

改了文件想更新很简单：

1. 仓库主页 → 进到要替换的文件所在位置
2. 点 **Add file** → **Upload files**，把新文件拖进去
3. 最下面 **Commit changes**
4. 等 1–2 分钟，刷新网站即可看到新版（手机如果没更新，下拉刷新或清一次缓存）

---

## 常见问题

**Q：打开链接是 404 / 空白？**
- 确认 Pages 里 Branch 选的是 `main` 和 `/ (root)`，且已 Save
- 确认上传的是文件夹**里的内容**，仓库根目录应能直接看到 `index.html`（不是 `combined-app/index.html`）
- 刚开通需等 1–2 分钟

**Q：图表不显示 / 云同步连不上？**
- 网站靠 CDN 加载图表库，首次需联网；大陆偶发慢，刷新即可
- 云同步要先填 LeanCloud 凭证（见 `部署指南.md` 第一步），不填则数据各设备独立

**Q：能换成好记的网址吗？**
- 可以买域名后在 Pages 的 **Custom domain** 绑定，属进阶操作，需要可以再帮你
